# PacketAlert

**PacketAlert** is a lightweight FiveM network warning HUD.  
It displays a bottom-center warning when the player's ping goes above the configured limit.

---

## Features

- Server-side ping detection
- Client-side HUD display
- Bottom-center warning text
- Ping displayed above the warning
- Configurable ping threshold
- Configurable text position
- Toggle command included
- Lightweight and easy to install

---
